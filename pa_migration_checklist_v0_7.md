# PA 防火牆汰換檢查表 (v0.7)

本檢查表供 PA 防火牆正式上線前、中、後之確認與查驗使用。

## 施工前
- [ ] 取得並確認 HPE-5130-01 (192.168.1.23) 唯讀設定檔，特別釐清 Port 12, 21, 48 及 Uplink 的 VLAN 狀態 (Access/Trunk/Hybrid) (目前待確認)。
- [ ] 確認 Cisco 9200 (24-port PoE) 的管理 IP、MAC、Uplink Port 及 PoE AP 對應的 Port (目前待確認)。
- [ ] 確認 1F 幼兒園廁所旁分機櫃的實體拓樸與設備型號 (目前待確認)。
- [ ] 確認 HPE-5130-03 的實體安裝位置 (目前待確認)。
- [ ] 釐清現行 FortiGate LAN1 與 LAN2 同時進入 5130-01 之 VLAN 隔離機制，確保防火牆移轉不產生 L2 迴圈。
- [ ] 針對 CCTV 監視器混入 LAN1 網段之風險，評估於 PA 上建立獨立 Zone 之可行性與架構。
- [ ] 確認 PA 上之路由設定能接手 Core01 Port16 的 VLAN2 (tagged) 與 VLAN3 (tagged, 192.168.3.71/24) 網段。

## 施工中
- [ ] 切換前，確認已標示所有實體網路線與對接之防火牆 Port。
- [ ] 依據《PA 防火牆介面對應表》依序切換 TANet/WAN、LAN1、LAN2 及 WiFi 線路。
- [ ] 進行設備切換時，監控 HPE-5130-01 與 Core01 狀態，避免廣播風暴發生。
- [ ] 若實施 CCTV 獨立 Zone，完成 PA 防火牆相關 VLAN 與資安策略實作。

## 施工後
- [ ] 驗證 LAN1 與 LAN2 原有設備連外網路功能是否正常。
- [ ] 驗證 WiFi (Cisco 9200 及 FortiAP) 無線網路服務是否正常且分配正確網段。
- [ ] 測試 Core01 下游高密度終端之連線：1F 分機櫃 (~101 MACs) 及 HPE-5130-03 (~129 MACs) 是否順暢上網。
- [ ] 驗證內部重要伺服器 Siraya Pi (192.168.1.12, 接於 5130-01 port 21) 之服務存取是否正常。
- [ ] 觀察 PA 防火牆 Log，確認無異常迴圈、非預期封包或阻擋正常業務流量的情況。
