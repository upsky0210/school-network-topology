# v0.8 待辦事項清單：分機櫃與支線補查

為完善後續 PA 防火牆管理與架構釐清，將於 v0.8 階段進行以下資訊之盤點 (本階段嚴守不更動設備設定之原則)：

## 1. 1F 幼兒園廁所旁分機櫃
- [ ] 實地確認機櫃安裝位置與供電狀況。
- [ ] 盤點該機櫃內之交換器廠牌、型號、數量與串接方式 (確認是否有再往 1F-4F 延伸)。
- [ ] 釐清由 Core01 port12 進入該機櫃的 Uplink port 接法。

## 2. HPE-5130-03 實體位置確認
- [ ] 追蹤 Core01 port9 的實體連線路徑。
- [ ] 找出並記錄 HPE-5130-03 實際安裝的機櫃與位置。
- [ ] 確認其與下游 ICX7150 等設備的對接 Port 資訊。

## 3. Cisco 9200 資訊補完
- [ ] 尋找 Cisco 9200 (24-port PoE) 的實體擺放位置。
- [ ] 查明該設備的管理 IP 與 MAC Address。
- [ ] 確認接回上游防火牆 (原 FortiGate port5) 之 Uplink port。
- [ ] 整理該交換器供電給各個 FortiAP 之 PoE port 對應關係。

## 4. 取得 HPE-5130-01 唯讀設定
- [ ] 追蹤《廠商唯讀資料請求書》進度，確認是否取得 5130-01 的 read-only 帳號或匯出檔。
- [ ] 解析設定檔，釐清 port 12, 21, 48 與 Uplink 之 VLAN 隔離模式 (Access / Trunk / Hybrid)。

## 5. FortiAP 對照表建立
- [ ] 配合 Cisco 9200 盤點，建立全校 FortiAP 總表。
- [ ] 總表應包含：AP 實體位置/名稱、MAC 位址、配發 IP 及對應之 Cisco 9200 port 號。
